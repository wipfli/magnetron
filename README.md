# magnetron
Collapse carriageways by making them magnetic

## Demo

https://wipfli.github.io/magnetron/

## Usage

Download the planetiler jar with:

```
wget https://github.com/onthegomap/planetiler/releases/latest/download/planetiler.jar
```

Then execute:

```
./run.sh
```
